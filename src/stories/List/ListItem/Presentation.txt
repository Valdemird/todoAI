

Pasos generales:

Planeación del componente
- Casos de uso  
    - Se usará para ver un el listado de tareas
    - Se usará para ver un listado de listas de tareas 
        - Ambos casos requieren:
            - Poder eliminar elementos de la lista
            - Poder ver el texto relacionado con cada elemento de la lista
            - Poder editar el texto relacionado con cada elemento de la lista
        - Diferencias:
            - El listado de tareas requiere un check que muestra el estado de la tarea
            - El listado de listas de tareas no requiere ese check    
Requerimientos
    - Debe recibir un listado de elementos
    - Debe mostrar un texto editable onClick por cada uno de los elementos de la lista
    - Debe tener un botón de eliminar que ejecute una callback
    - Debe tener un checkbox que se renderice condicionalmente
    - El checkbox y el texto editable comparten la misma callback
